В Title пишите :

Bug: Кратко о чем баг

В описание опишите как вы его получили 

Вот пример :
```
## Описание
После первой переноски здания его меш “запоминается” и затем отображается при последующих переносах других зданий. В результате при переносе показывается меш не того здания.

## Шаги по воспроизведению
1.Ставим два разных здания на сетку 
2.Нажимаем на любое здание и переносим его
3.Нажимаем на другое здание и видим что меш используется не тот
```
Желательно использовать скрины
